# 網站結構重整建議

> 針對 https://www.chtwnews.com/ 現有內容分析與重組方案

---

## 一、現況問題

### 1. 分類混亂，文章歸屬重疊
- 「帳號實名 支付代購」塞了 14 篇文章，涵蓋註冊、購物、代購、手機號碼，主題跨度太大
- 「跨境電商」混入公眾號出租、淘寶帳號出租，與「帳戶出租」分類功能重疊
- 「帳戶出租」分類存在但沒有文章，所有出租相關內容散落在其他分類
- 同一主題（如小紅書經營）在「新媒體運營」和「帳號實名」都有文章，使用者不知道去哪找

### 2. 內容薄弱且重複
- 多篇文章角度高度重複（如小紅書專業號投流有 3 篇幾乎相同的切入點）
- 每篇文章只有一段摘要，缺乏深度內容
- 全站僅 32 篇短文，卻分散在 7 個分類 + 分頁，導致多數頁面看起來空蕩

### 3. 導航體驗差
- 「帳號實名 支付代購」需要翻 2 頁才能看完，但其實只有 14 篇
- 分類名稱不直覺（「帳號實名 支付代購」太長、「帳戶出租」與「跨境電商」界線不清）
- 沒有清楚的服務導向頁面，使用者難以判斷「我該找哪個服務」

## 二、建議網站架構（4 大分類 + 服務頁）

```
首頁
├── 服務項目（獨立頁面，非文章列表）
│   ├── 帳號代辦與租用
│   ├── 代購與驗貨
│   ├── 公司註冊與銀行開戶
│   └── 新媒體代營運
│
├── 教學指南（文章中心）
│   ├── 帳號註冊
│   ├── 支付工具
│   ├── 新媒體經營
│   └── 跨境電商
│
├── 關於我們
│
└── 聯繫我們
```

## 四、內容數量對比

| | 現況 | 建議 |
|---|---|---|
| 分類數 | 7 個（含空分類） | 4 個教學子分類 + 4 個服務頁 |
| 文章數 | 32 篇短文（大量重複） | 16 篇深度文章（整併去重） |
| 服務頁 | 無（全靠文章列表） | 4 個獨立服務頁（有 CTA） |
| 分頁 | 有（「帳號實名」分 2 頁） | 無需分頁 |

## 五、執行優先順序

1. **先做服務頁** — 這是轉換的核心，使用者來是要找服務的，不是看文章
2. **合併重複文章** — 32 篇 → 16 篇，每篇加深內容到 800-1500 字
3. **重建首頁** — 從文章列表改為服務導向的 Landing Page
4. **新增關於我們** — 建立信任感
5. **最後優化 SEO** — 統一標題格式、補 meta description
